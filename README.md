# CRM Pipeline Analytics System

This project simulates a CRM analytics pipeline that transforms raw People, Company, and Deal datasets into structured analytical data for revenue and pipeline monitoring.

## Technologies Used
SQL
Data Modeling
Palantir Foundry
Analytics Dashboards

## Data Model
Industry → Company → Person → Deal → Revenue Metrics

## Key Transformations
- People ID generation using window functions
- Phone number cleaning and standardization
- Company name normalization
- Duplicate record removal
- Revenue tier creation
- Pipeline KPI calculation

## Business Metrics
- Open Pipeline Revenue
- Closed Won Revenue
- Win Rate
- Top Revenue Companies
- Industry Level Performance
